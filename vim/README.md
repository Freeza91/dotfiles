# dotfiles-myvim-settings

### create!
```sh
git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/plugin/Vundle.vim && \
cp -f ./vundle.vim ~/.vim/vundle.vim && \
cp -f ./vimrc ~/.vimrc && \
vim ~/.vim/vundle.vim
```

### update
```
cp -f ./vundle.vim ~/.vim/vundle.vim && \
cp -f ./vimrc ~/.vimrc && \
vim ~/.vim/vundle.vim
```

just ```PluginInstall``` or ```BundleInstall```
